git
是一个版本管理工具，是可以在电脑不联网的情况下，只在本地使用的一个版本管理工具，其作用就是可以让你更好的管理你的程序
github
是一个网站，每个程序员自己写的程序可以在github上建立一个网上的仓库，每次提交的时候可以把代码提交到网上，这样每次提交别人也都可以看到你的代码，同时别人也可以帮你修改你的代码，这种开源的方式非常方便程序员之间的交流。

上传：
在库中点击Upload files 点击cloose your files,选择要上传的文件；
同步：
将本地同步到平台：
在本地中找到和平台的库相关联的文件夹，右击文件夹选择Git GUI Here,经过查找更新，在Unstaged Changes框中会显示本地与平台库中不同的文件，左击文件的小图标，文件会在Staged Changes框中出现，双击文件，文件的相关内容会在Staged for commit框中展现，这两个操作都是为了将工作区的文件传到暂存区，这之后要在Commit Message框中点击commit，在出现的命令框中输入命令git clone 地址（地址是从要同步的库中Clone or download中复制来的，注意，粘贴到命令行中时只能右击点粘贴不能用快捷键），回车，等命令执行完关掉命令页面，点击Push,在弹出的页面加载完之后结束。
将平台同步到本地：
在本地中找到和平台的库相关联的文件夹，右击文件夹选择Git GUI Here,点击Remote,点击fetch from中的origin，弹出的小框显示成功后close,此时文件从远程仓库即平台仓库传到本地仓库，点击marge,点击Local marge...,文件即从本地仓库传到工作区即本地。
